**Beauty Parlour Web Portal**
A full-stack web application for managing appointments, services, and customer engagement for a beauty salon.

**Features**
✅ Secure User Authentication - Registration, login, and role-based access for clients and admins
✅ Dynamic Service Catalog - Add, edit, and manage beauty services with images and pricing
✅ Real-time Appointment Booking - View available slots and book appointments with email confirmation
✅ Client Feedback System - Ratings and reviews for services
✅ Admin Dashboard - Comprehensive management of appointments, services, clients, and payments
✅ Payment Integration - Secure payment processing with multiple methods
✅ Fully Responsive Design - Works seamlessly across all devices

**How It Works**
1. Clients register/login to access the portal
2. Browse services with details, pricing, and availability
3. Book appointments by selecting preferred date, time, and beautician
4. Make secure payments online
5. Provide feedback and ratings after service
6. Admins manage all operations through a centralized dashboard

**Tech Stack**
**Frontend:** React.js, HTML5, CSS3, JavaScript (ES6), Bootstrap/Tailwind CSS
**Backend:** Node.js, Express.js
**Database:** MongoDB
**Authentication:** JWT (JSON Web Tokens)
**Other Tools:** Mongoose, Nodemailer, Cloudinary, Postman

**Installation**
1. Clone the repository:
git clone https://github.com/your-repo/beauty-parlour-portal.git

2. Install backend dependencies:
cd backend
npm install

3. Install frontend dependencies:
cd ../frontend
npm install

4. Set up environment variables (create .env files in both backend and frontend folders)

5. Run the development servers:
# Backend server
cd backend
npm start

# Frontend server (in another terminal)
cd frontend
npm start
